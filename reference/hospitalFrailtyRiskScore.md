# Hospital Frailty Risk Score data set

Hospital Frailty Risk Score data set

## Usage

``` r
hospitalFrailtyRiskScore
```

## Format

### `hospitalFrailtyRiskScore`

A data frame with 109 rows and 4 columns:

- concept_set:

  The name of the concept set to be provided

- points:

  Number of points associated with the codelist

- icd10_code:

  ICD-10 code used

- icd_description:

  Name of the ICD-10 code
